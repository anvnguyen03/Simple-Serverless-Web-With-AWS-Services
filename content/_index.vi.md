# Xây dựng một trang web quản lý sản phẩm không máy chủ đơn giản với DynamoDB, Lambda, API Gateway và S3

### Tổng quan

 Trong hướng dẫn này, chúng ta sẽ tìm hiểu cách tạo một trang web không có máy chủ đơn giản bằng các dịch vụ AWS như DynamoDB, Lambda, API Gateway và S3. Ứng dụng sẽ bao gồm các tính năng CRUD (Tạo, đọc, cập nhật, xóa) để quản lý sản phẩm. Với cách tiếp cận không có máy chủ, chúng ta sẽ không cần quản lý máy chủ, cho phép chúng ta tập trung vào việc xây dựng ứng dụng của mình trong khi giữ chi phí thấp và mở rộng dễ dàng.
